Anonymous Bitcoin Club

Active since January 1, 2021 · Category: Bitcoin education, open-source tools, and culture

What is Anonymous Bitcoin Club?

Anonymous Bitcoin Club (ABC) is a cypherpunk-leaning collective focused on building Bitcoin-first tools, live experiences, and educational content. The project spans websites, games, live music, and on-chain artifacts that aim to make Bitcoin culture more accessible and fun.

Current Focus Areas

  • Maintaining and expanding the AllAboutBTC.com knowledge base.
  • Experimenting with Bitcoin-adjacent games, simulators, and educational tools.
  • Running community events, live performances, and collaborative builds.

How to Get Involved

Contributions are welcome via pull request to the AnonymousBitcoinClub/collective-website repository or by joining community discussions linked from the main sites.
